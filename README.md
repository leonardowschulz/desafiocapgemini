# desafiocapgemini

- Projeto de 3 questões feito para o Desafio de Programação da Academia Capgemini

- A primeira questão imprime na tela uma escada, onde o usuário informa um número, que serve para informar ao sistema quantas linhas e caracteres por linha a escada vai ter;

- A segunda questão é um sistema de conferência de senhas. O usuário digita uma senha e o sistema confirma se ela é uma senha segura, ou quais características devem ser adicionadas para ela ficar segura;

- A terceira questão analisa palavras e textos procurando anagramas entre elas, ao final do programa ele informa a quantidade de anagramas possíveis com o texto.

-----------------------------

Para instalação é necessário baixar a pasta clicando em 'Code', 'Download zip', descompactar os arquivos, instalar o Java Development Kit, abrir a pasta do projeto então é possível executar individualmente cada uma das 3 questões.
